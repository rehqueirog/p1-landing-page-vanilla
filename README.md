# Portfolio Vanilla

Landing page profissional construída com HTML, CSS e JavaScript puro — sem frameworks, sem bibliotecas.

---

## Tecnologias

- HTML5 semântico
- CSS3 com design tokens (custom properties)
- JavaScript puro (DOM, eventos, accordion)
